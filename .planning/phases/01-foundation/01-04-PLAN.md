```python
import logging
from contextlib import asynccontextmanager

import uvicorn
from fastapi import FastAPI

from akc.core.config import settings
from akc.patterns.store import JsonlStore

# ---------------------------------------------------------------------------
# Logging
# ---------------------------------------------------------------------------
logging.basicConfig(
    level=logging.INFO,
    format="%(asctime)s %(name)s %(levelname)s %(message)s",
)
logger = logging.getLogger("akc")

# ---------------------------------------------------------------------------
# Store — module-level singleton, instantiated after settings validated
# ---------------------------------------------------------------------------
store = JsonlStore(kb_dir=settings.akc_kb_dir)

# ---------------------------------------------------------------------------
# Lifespan — runs at startup and shutdown (not @app.on_event — deprecated)
# ---------------------------------------------------------------------------

@asynccontextmanager
async def lifespan(app: FastAPI):
    stats = await store.load_stats()
    logger.info(
        "AKC starting — KB_DIR: %s, patterns: %d",
        settings.akc_kb_dir,
        stats["total"],
    )
    yield
    logger.info("AKC shutting down")


# ---------------------------------------------------------------------------
# App factory
# ---------------------------------------------------------------------------
app = FastAPI(
    title="AKC — Agent Knowledge Collective",
    version="0.1.0",
    lifespan=lifespan,
)


# ---------------------------------------------------------------------------
# Health endpoint — FNDTN-02
# Required by AgentBase: GET /health → {"status": "ok", "pattern_count": N}
# ---------------------------------------------------------------------------

@app.get("/health")
async def health():
    stats = await store.load_stats()
    return {"status": "ok", "pattern_count": stats["total"]}


# ---------------------------------------------------------------------------
# Entry point
# ---------------------------------------------------------------------------

if __name__ == "__main__":
    uvicorn.run("main:app", host="0.0.0.0", port=8080, reload=False)
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| HTTP client → GET /health | Unauthenticated endpoint; response contains only pattern count (no PII, no secrets) |
| module import → akc.core.config | settings singleton instantiated at module load; ValidationError must propagate uncaught |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | Information Disclosure | GET /health response | accept | Response contains only {"status": "ok", "pattern_count": N} — no credentials, no file paths, no internal state. Low sensitivity. |
| T-04-02 | Information Disclosure | lifespan logger.info | mitigate | Log only settings.akc_kb_dir and stats["total"]. NEVER log settings.llm_api_key, settings.llm_model, or the full settings object. The startup log line uses "%s, patterns: %d" with only kb_dir and count. |
| T-04-03 | Denial of Service | uvicorn accepting connections during broken startup | mitigate | settings = Settings() at module level means pydantic.ValidationError fires before uvicorn binds to port 8080. Process crashes before the health endpoint is reachable. |
| T-04-04 | Spoofing | /health returns stale count | accept | load_stats() reads from disk on every request (no in-memory cache in Phase 1). Count is always fresh. Caching is a Phase 3 concern. |
</threat_model>
