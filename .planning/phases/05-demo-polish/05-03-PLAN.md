## threat_model

**ASVS L1 Threat Assessment**

| Threat | Vector | Mitigation |
|--------|--------|------------|
| Credential exposure in README | LLM_API_KEY shown in curl examples | All examples use placeholder values (`LLM_API_KEY=your-api-key`); no real credentials in README |
